Method & Fidelity

Orbital Propagation

SGP4 / SDP4 (Simplified General Perturbations 4) — the industry-standard analytic model for near-Earth satellites. Implemented via satellite.js (Vallado 2006 / Celestrak). Accounts for J2 oblateness, atmospheric drag (via B* drag term in TLE). Valid range: 200–2000 km circular orbits, short-arc predictions (days). Error: typically <1 km position error over 1–2 day arcs with fresh TLE.

Link Budget

  • EIRP = P_tx + G_tx − L_line — effective isotropic radiated power
  • FSPL = 92.45 + 20·log₁₀(d_km) + 20·log₁₀(f_GHz) — ITU-R P.525-3 free-space path loss
  • C/N₀ = EIRP − FSPL − L_other + G/T + 228.6 — standard RF link equation
  • Eb/N₀ = C/N₀ − 10·log₁₀(R_b) — bit energy to noise density
  • Margin = Eb/N₀ − Required Eb/N₀ — positive = link closes

Geometry

Slant range computed from spherical-Earth geometry (R_e = 6371 km) using satellite altitude and elevation angle. Footprint radius via the rigorous Earth-central-angle formula ρ = acos((R_e/(R_e+h))·cos(el)) − el — the trailing − el is the local-horizon correction; omitting it (geocentric approximation) overstates the radius by a few percent at low mask angles. Accurate to <0.5% vs WGS-84 ellipsoid at LEO altitudes.

Loss Allowances (Simplified)

  • Atmospheric/gaseous: Per-band zenith allowance (UHF 0.1 dB, S 0.2 dB, X 0.3 dB, Ka 0.8 dB) scaled by slant-path airmass ≈ 1/sin(el) (clamped at 5° elevation to bound the horizon singularity), so low-elevation passes lose meaningfully more. Approximates ITU-R P.676; use the full model for operational analysis.
  • Rain attenuation: Conservative zenith allowance (UHF 0 dB, S 0.1 dB, X 0.5 dB, Ka 2.0 dB) likewise scaled by ≈ 1/sin(el) airmass. Actual depends on region and climate zone. Use ITU-R P.618 for fade-margin sizing.
  • Pointing & polarization: Fixed 0.5 dB combined. Actual depends on antenna system.

Pass Scheduling

Access windows computed by propagating SGP4 at configurable time steps, computing look angles (azimuth, elevation, range) to each ground station using Leaflet/satellite.js ecfToLookAngles, and recording intervals where elevation ≥ minimum mask angle.

Standard link-budget and SGP4 access math — same physics as reference tools (STK/MATLAB), lighter and owned. Rain and atmospheric models use zenith allowances scaled by ~1/sin(el) airmass; verify against ITU-R P.618 / P.676 for operational use. TLE accuracy degrades beyond 1–2 days from epoch — refresh TLEs for mission-critical scheduling.
